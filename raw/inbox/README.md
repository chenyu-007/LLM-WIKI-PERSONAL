# inbox

这里放外部知识库批量导出的待处理资料，例如网易文档、有道云笔记、网盘资料或其他历史文件。

## 规则

- 这里的文件保持原样，不直接改写。
- AI 可以读取、分类、去重、提取候选信息和生成审核建议。
- 未经用户明确批准，AI 不得把这里的内容写入正式 `wiki/` 页面。
- 用户批准后，AI 需要回到原始文件校验关键事实，再录入 `wiki/`。

## 建议流程

1. 把导出的文件放进本目录，尽量保留原文件夹结构。
2. AI 生成候选清单：文件名、主题、价值判断、风险、建议去向。
3. 用户逐条或批量审核。
4. AI 对通过项做事实校验。
5. AI 写入 `wiki/`，并更新 `index.md` 和 `log.md`。
