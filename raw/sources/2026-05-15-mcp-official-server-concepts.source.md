# MCP 官方 Server Concepts

## 来源说明

这是 Model Context Protocol 官方文档的服务器概念页，用于确认 MCP Server 如何通过 Tools、Resources 和 Prompts 向 AI 应用暴露能力。

## 本 Wiki 中的用途

- 用于定义 Tools、Resources、Prompts 三类核心 server features。
- 用于说明三类能力分别由模型、应用和用户主导。
- 用于支持后续 MCP Server 设计、知识库接入和 Agent 工具边界讨论。
