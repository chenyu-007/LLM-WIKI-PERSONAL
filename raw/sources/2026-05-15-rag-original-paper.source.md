# Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks

## 来源说明

这是 RAG（Retrieval-Augmented Generation）概念的经典论文来源，用于在对比 MCP 和 RAG 时给出外部依据。

## 本 Wiki 中的用途

- 用于说明 RAG 将参数化模型与外部非参数化记忆或检索结果结合。
- 用于支撑 `MCP vs RAG` 页面中对 RAG 边界的描述。
- 用于避免把 MCP 和 RAG 的差异写成未经来源支持的个人判断。
