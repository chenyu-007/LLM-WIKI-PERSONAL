# Browserbase Docs

## 来源说明

这是 Browserbase 官方「What is Browserbase」文档，用于确认 Browserbase 的平台定位、核心能力和与 Stagehand 的关系。

## 本 Wiki 中的用途

- 用于记录 Browserbase 作为面向浏览网页 Agent 的云浏览器和 Web 交互平台。
- 用于说明 Browserbase 不只是浏览器会话，还包含 search、fetch、functions、model gateway、identity 和 observability 等平台能力。
- 用于区分 Browserbase 和 Tavily：Browserbase 更偏真实浏览器交互，Tavily 更偏搜索和内容获取。
