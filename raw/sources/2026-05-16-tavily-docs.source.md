# Tavily Docs

## 来源说明

这是 Tavily 官方文档入口和 API Reference，用于确认 Tavily 的产品定位、API 能力和在 AI Agent / RAG 系统中的位置。

## 本 Wiki 中的用途

- 用于记录 Tavily 作为面向 AI 应用的搜索、抽取、爬取、站点地图和研究 API。
- 用于区分 Tavily 和 Browserbase：Tavily 更偏「搜索和内容获取」，Browserbase 更偏「浏览器会话和网页交互」。
- 用于支撑后续讨论 Tavily 是否适合接入 MCP、RAG 或 Agent 工作流。
