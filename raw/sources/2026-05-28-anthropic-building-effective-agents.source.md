## 来源说明

这是 Anthropic 官方 Agent 方法论文章，讨论他们从客户和内部实践中总结出的 Agent 架构经验。

## 本 Wiki 中的用途

- 作为 `AI Agent` 和 `AI 工作流` 页面对 Anthropic 方法论的官方校验来源。
- 用于区分 workflow 与 agent：固定、可预测流程优先；真正需要自主决策时再使用 agent。
- 用于支撑“简单、可组合模式优先于复杂框架”的本地维护原则。
