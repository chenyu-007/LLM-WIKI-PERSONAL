# Effective context engineering for AI agents

## 来源说明

这是 Anthropic 官方关于 Agent 上下文工程的文章，讨论如何管理有限上下文，让 Agent 在长任务中保持有效状态。

## 本 Wiki 中的用途

- 作为 `上下文工程` 概念页的官方来源。
- 用于支撑本地 LLM Wiki 的核心价值：把需要长期保留的上下文外化为可维护文件，而不是全部塞进对话窗口。
- 用于校验 compaction、structured memory、just-in-time retrieval、subagent 等实践。
