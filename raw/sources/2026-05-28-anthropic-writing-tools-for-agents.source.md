# Writing effective tools for AI agents — with agents

## 来源说明

这是 Anthropic 官方关于 Agent 工具设计和评估的工程文章，重点是如何让工具更容易被 Agent 正确选择、调用和组合。

## 本 Wiki 中的用途

- 用于校验 MCP 工具、CLI 工具和本地脚本应如何设计给 Agent 使用。
- 用于支撑“工具命名、返回信息、评估循环、工具数量控制”这些维护规则。
- 用于后续设计 YoudaoNote、Douyin、Bilibili、小红书等采集工具时作为工程基准。
