# Anthropic

## 是什么

Claude 的开发商，AI Agent 方法论的重要推动者。

## 关键贡献

- **Claude Code** — 终端 AI 编程助手，上下文工程先驱。
- **Building Effective AI Agents** — Agent 设计方法论：从简单开始、逐步加复杂度。
- **Cat Wu (产品经理)** — 提出"PRD 已死，原型万岁"的 AI 产品开发范式。
- **Agent Architecture** — 强调单体 Agent 起步，拒绝过早引入多 Agent 复杂度。

## 核心哲学

> 从能工作的事情开始，只在实际需要时增加复杂度。

这也反映在他们的 Agent 设计指南里：Skill 系统、上下文管理、工具调用——都遵循"够用就好"的原则。
