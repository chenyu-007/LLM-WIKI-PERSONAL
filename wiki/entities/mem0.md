# Mem0

## 是什么

轻量级 Agent 长期记忆工具。核心理念：**不存对话，存值得记住的东西**。

## 关键特性

- 自动过滤噪声——用"重要性×时效性×相关性"公式判断什么值得记。
- 压缩存储——不是原文，是结构化摘要。
- 适合单 Agent 和多 Agent 场景。
