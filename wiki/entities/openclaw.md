# OpenClaw

## 是什么

OpenClaw 是一个开源的 AI Agent 框架/平台，支持多 Agent 协作、Skill 系统、知识库管理。

## 关键能力

- **Skill 系统** — 安装和管理可复用的 Agent Skill。
- **多 Agent 共用记忆** — 不同 Agent 之间共享上下文和记忆。
- **飞书集成** — 官方插件支持飞书，扩展企业协作场景。
- **开源** — 社区可定制和扩展。

## 相关来源

- 飞书官方插件让 OpenClaw 的玩法扩展到企业办公场景。
- 支持安装第三方 Skill，小白也能快速上手。
- 多 Agent 共用记忆方案解决"不同 Agent 各自失忆"的问题。
