# AWS RAG vs Fine-tuning 对比

## 来源

- 作者：AWS Prescriptive Guidance
- 发布时间：未标明
- URL：https://docs.aws.amazon.com/prescriptive-guidance/latest/retrieval-augmented-generation-options/rag-vs-fine-tuning.html
- 本地来源记录：`raw/sources/2026-05-15-aws-rag-vs-fine-tuning.source.md`

## 核心内容

- RAG 适合连接会变化的外部文档和知识源，并保留来源追溯能力。
- Fine-tuning 适合让模型更符合组织风格、任务格式或稳定领域行为。
- 二者不是互斥关系，可以在同一个系统中组合使用。
