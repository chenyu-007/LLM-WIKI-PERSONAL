# Browserbase Docs

## 来源

- 作者：Browserbase
- 发布时间：未标明

## 核心内容

- Browserbase 官方文档把它定义为构建和部署能像人一样浏览、交互网页的 Agent 的完整平台。
- Browserbase 提供 headless browsers、web search、page fetching、functions、model access 和 identity。
- Browserbase 的浏览器能力包括可扩展的 headless browser fleets、隔离会话和全球基础设施。
- Search 和 Fetch 是浏览器会话的快速、token-efficient 补充，适合直接搜索和获取网页内容。
- Stagehand 是 Browserbase 生态中的 AI-native browser automation SDK，结合 Playwright 级控制和 `act`、`extract`、`observe` 等 AI 原语。
