# MCP 官方介绍

## 来源

- 作者：Model Context Protocol
- 发布时间：未标明
- URL：https://modelcontextprotocol.io/docs/getting-started/intro
- 本地来源记录：`raw/sources/2026-05-15-mcp-official-intro.source.md`

## 核心内容

- MCP 是连接 AI 应用与外部系统的开放标准。
- MCP 的连接对象包括数据源、工具和工作流。
- 官方介绍用 USB-C 作类比：MCP 的目标是为 AI 应用连接外部能力提供统一接口。
- MCP 的价值因角色不同而不同：开发者减少集成复杂度，AI 应用获得更多外部能力，终端用户获得更能访问数据和执行任务的助手。
